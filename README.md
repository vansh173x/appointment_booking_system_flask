# 🏥 E-Healthcare Appointment Booking System

A web-based healthcare appointment booking system built using **Flask, HTML, CSS, JavaScript, and SQLite**. It allows patients to book appointments, doctors to manage appointments, and provides a secure and user-friendly healthcare platform.

## ✨ Features

- Patient & Doctor Registration/Login
- Book and Manage Appointments
- Doctor Profile Management
- Nearby Doctor Search
- Patient-Doctor Chat
- Healthcare Subscription Plans
- UPI Payment Integration
- Secure Authentication

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Flask (Python)
- **Database:** SQLite

## 🚀 Installation

```bash
git clone https://github.com/your-username/e-healthcare-appointment-system.git
cd e-healthcare-appointment-system
pip install flask
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
